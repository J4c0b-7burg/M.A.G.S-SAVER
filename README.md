# M.A.G.S Saver

Store you favorite Movies, Anime, Games, and TV shows with M.A.G.S. Perfect to store your favorites to recommend to your friends later!

### Frontend

- React
- HTML
- Javascript
- CSS

### Backend

- Bit.io
- Python
- Jango
